Comandos ejecutados:
git worktree add -b feature-b ../wt-feature-b main
git worktree list

/home/ronny/Documents/dev/wt-feature-b                                                    7c2eac4 [feature-b]

Explicación:
La diferencia es que 'git worktree add -b' crea la rama y el directorio de trabajo en un solo paso,
mientras que con 'git branch' tendrías que crear la rama primero y luego añadir el worktree o hacer checkout.
Esto agiliza mucho el inicio de una nueva tarea aislada.
